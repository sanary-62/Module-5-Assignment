Question-1: What are the different ways to select an element in the DOM?

Answer: getElementById("id"), getElementsByClassName("class"), getElementsByTagName("tag"),querySelector("selector"), querySelectorAll("selector")

Question-2: What is the difference between innerHTML,innerText and innerContent

Answer: innerHTML sets HTML content includes tags, innerText sets visible text excludes hidden elements, textContent sets all text includes hidden elements, excludes HTML tags

Question-3: What is event delegation in the DOM?

Answer: Event Delegation is a method that attach an event listener to a parent and handle child events using event bubbling

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is a event on a child that moves up to its parent
